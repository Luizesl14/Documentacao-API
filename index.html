Status Search status of the simulation and also the proposal

Get status by simulation id:
/external/simulations-status/{id}

Get status by proposal id:
/external/proposal-status/{id}

PARAMETERS


Name Description
Id *required Simulation Id

RESPONSES

Code Description
200 Successfully retrieved object
401 You are not authorized to view the resource
403 Accessing the resource you were trying to reach is forbidden
404 The resource you were trying to reach is not found
